# Wordly - Interactive Dictionary SPA

Wordly is a modern Single Page Application (SPA) designed to help users search for word definitions, pronunciations, and synonyms instantly. This project was built as part of a Summative Lab to demonstrate proficiency in API integration, DOM manipulation, and asynchronous JavaScript.

##  Features
* Live Search:** Fetch real-time data from the Free Dictionary API.
* Audio Playback:** Listen to the correct pronunciation of words.
* Favorites List:** Save words to your local storage to view them later.
* Responsive Design:** A clean, user-friendly interface that works on mobile and desktop.
* Error Handling:** Graceful handling of missing words or API connection issues.

## Built With
* HTML5:** Semantic structure for accessibility.
* CSS3:** Custom styling with CSS variables and animations.
* JavaScript (ES6+):** Fetch API, LocalStorage, and DOM manipulation.
